# CMP 슬러리 및 패드 교체 절차

## 1. 패드 교체 절차 (SOP-CMP-PAD-001)

### 1.1 교체 판단 기준

패드 교체 판단 기준이다.

| 기준 | 임계값 | 측정 방법 |
|------|--------|----------|
| 사용 시간 | > 500시간 | 장비 로그 |
| 패드 두께 | < 1.0 mm | 두께 게이지 |
| MRR 저하 | > 15%(신품 대비) | Test wafer |
| WIWNU 악화 | > 5% | Test wafer |
| 표면 상태 | glazing, groove 마모 | 육안 검사 |

### 1.2 필요 도구 및 자재

패드 교체에 필요한 도구와 자재이다. 신품 Polishing Pad(IC1000/SubaIV 또는 동등 사양), PSA(Pressure Sensitive Adhesive) 접착 패드, IPA(Isopropyl Alcohol) 세정액, 클린 와이퍼, 패드 두께 게이지, 고무 롤러(기포 제거용), 보호 장갑 및 보안경.

### 1.3 교체 절차 Step 1: 준비 (10분)

패드 교체 준비 단계이다. (1) 장비를 PM 모드로 전환 → (2) 슬러리 공급 중단 → (3) DI water로 플래튼 및 패드 세정(5분) → (4) 플래튼 회전 정지 → (5) 캐리어 헤드를 대기 위치로 이동. LOTO 절차를 실행하여 안전을 확보한다.

### 1.4 교체 절차 Step 2: 기존 패드 제거 (15분)

기존 패드 제거 절차이다. (1) 패드 한쪽 모서리에서 PSA 접착면 분리 시작 → (2) 천천히 당겨 패드를 플래튼에서 분리(급격히 당기면 PSA 잔여물 발생) → (3) 플래튼 표면의 PSA 잔여물을 IPA로 세정 → (4) 클린 와이퍼로 플래튼 표면 완전 건조 → (5) 플래튼 표면 이물질 및 손상 확인.

### 1.5 교체 절차 Step 3: 신품 패드 부착 (15분)

신품 패드 부착 절차이다. (1) 신품 패드의 PSA 보호 필름 제거 → (2) 패드를 플래튼 중심에 정렬하여 위치 → (3) 중심에서 바깥 방향으로 롤러를 사용하여 부착(기포 방지) → (4) 패드 에지가 플래튼 에지와 정확히 일치하는지 확인 → (5) 전체 면적의 접착 상태 확인(들뜸 없음).

### 1.6 교체 절차 Step 4: 패드 초기화 (30분)

패드 Break-in 절차이다. (1) 컨디셔너로 패드 표면 초기화(Downforce 7 lbf, 시간 20분, 플래튼 60 rpm, DI water 200 ml/min) → (2) Break-in 후 패드 표면 상태 확인(균일한 거칠기 형성) → (3) 슬러리 공급 시작.

### 1.7 교체 절차 Step 5: Qualification (30분)

패드 교체 후 Qualification 절차이다. (1) 더미 웨이퍼 5매 처리(패드 안정화) → (2) Test wafer 3매 연마 후 측정 → (3) 합격 기준: MRR 목표값 ± 10%, WIWNU < 5%, 스크래치 0개 → (4) 측정 결과 기록 및 승인. 불합격 시 추가 Break-in 또는 파라미터 조정.

### 1.8 패드 교체 후 주의사항

패드 교체 후 주의사항이다. 초기 10매는 Zone 압력을 5% 낮게 설정한다(새 패드의 높은 MRR 보상). 초기 50매는 매 10매마다 MRR 추이를 모니터링한다. 초기 2일간 강화 컨디셔닝 레시피를 적용한다(Downforce +1 lbf).

## 2. 슬러리 교체 절차 (SOP-CMP-SLR-001)

### 2.1 교체 판단 기준

슬러리 교체 판단 기준이다.

| 기준 | 임계값 | 측정 방법 |
|------|--------|----------|
| 잔량 | < 10% | 탱크 레벨 센서 |
| 사용 기간 | > 72시간(개봉 후) | 개봉 일시 라벨 |
| pH 변화 | 초기값 대비 ±0.5 | pH meter |
| Particle size | D50 기준 ±20% | Particle analyzer |
| MRR 저하 | > 10% | Test wafer |

### 2.2 슬러리 탱크 교체 절차

슬러리 탱크 교체 절차이다. 준비: 신품 탱크를 30분 전 반입(온도 안정화), 교반 확인, 라벨(타입/로트/유효기간) 확인. 배출: 장비 Idle 확인 → 공급 밸브 차단 → 라인 DI water 플러시 5분 → 기존 탱크 분리 → 잔량 기록 후 폐슬러리 처리. 연결: 신품 탱크 설치 → 라인 연결(퀵 커플링) → 누수 확인 → 개봉 일시 라벨 부착 → 교반기 가동.

### 2.3 프라이밍 및 검증

슬러리 교체 후 프라이밍 및 검증 절차이다. (1) 슬러리 펌프 프라이밍(공기 제거) → (2) 유량 확인(설정값 ± 5%) → (3) 슬러리 pH 측정(스펙 이내 확인) → (4) Test wafer 1매 연마 후 MRR 확인 → (5) 교체 기록 작성(탱크 로트 번호, 교체 일시, 측정값).

### 2.4 슬러리 라인 플러시 절차

정기 또는 슬러리 타입 변경 시 라인 플러시 절차이다. (1) 슬러리 공급 중단 → (2) DI water 밸브 개방 → (3) DI water 플러시(500 ml/min × 10분) → (4) 배출수가 투명해질 때까지 계속 → (5) 슬러리 타입 변경 시 추가 플러시 10분 → (6) 밸브 전환 후 신규 슬러리 공급.

### 2.5 슬러리 관리 주의사항

슬러리 관리 주의사항이다. 온도 관리: 보관 온도 15~25°C. 유효 기간 엄수: 개봉 후 72시간 초과 사용 금지. 혼합 금지: 서로 다른 로트의 슬러리 혼합 금지. 결정화 방지: 슬러리 노출 부위 정기 세정. 폐슬러리 처리: 환경 규정에 따라 지정 용기에 수집.

## 3. 컨디셔너 디스크 교체

### 3.1 교체 판단 기준

컨디셔너 디스크 교체 판단 기준이다. 사용 시간 > 200시간. 다이아몬드 입자 탈락(육안 + 현미경 확인). Pad conditioning 후에도 MRR 회복 안 됨. 패드 표면에 비정상 스크래치 패턴 발생. 알람 A200(컨디셔너 이상) 반복 발생.

### 3.2 교체 절차

컨디셔너 디스크 교체 절차이다. (1) 컨디셔너 Arm을 정비 위치로 이동 → (2) 디스크 고정 나사 분리(3~4개) → (3) 기존 디스크 제거, 사용 시간 기록 → (4) 마운트 표면 IPA 세정 → (5) 신품 디스크 장착, 나사 균등 토크로 체결 → (6) Arm Sweep 동작 확인 → (7) Break-in conditioning 5분 → (8) Test wafer로 MRR 및 스크래치 확인.

## 4. Retaining Ring 교체

### 4.1 교체 판단 기준

Retaining Ring 교체 판단 기준이다. 두께 < 1.5mm(초기 두께 대비). 불균일 마모(편마모). 웨이퍼 에지 WIWNU 악화. 알람 A123(Head Pressure Out of Range) 반복 발생. 알람 A125(Retaining Ring 압력 이상) 발생.

### 4.2 교체 절차

Retaining Ring 교체 절차이다. (1) 캐리어 헤드를 정비 위치로 이동 → (2) 헤드 하부의 Ring 고정 볼트 분리 → (3) 기존 Ring 제거, 두께 4점 측정 기록 → (4) 헤드 마운트 면 세정 → (5) 신품 Ring 장착, 볼트 균등 토크 체결(15 N·m) → (6) Pressure Hold Test 실행 → (7) Test wafer 3매 연마, MRR/WIWNU/에지 프로파일 확인.

## 5. Membrane 교체 절차

### 5.1 교체 판단 기준

Membrane 교체 판단 기준이다. Pressure Hold Test 불합격(30초간 압력 강하 > 0.3 psi). Zone 압력 진동 발생(알람 A127). 육안 검사에서 균열, 변색, 박리 발견. 사용 기간 > 3개월(분기 PM 시 교체 권장). WIWNU 악화 원인이 멤브레인으로 특정된 경우.

### 5.2 교체 절차 (Step-by-step)

Membrane 교체 절차이다. (1) 캐리어 헤드를 정비 대에 거치 → (2) Retaining Ring 분리 → (3) 기존 Membrane을 헤드 본체에서 분리(클립 또는 나사 방식) → (4) 헤드 본체 Air Channel 통기 확인(각 Zone별 에어 분사) → (5) 헤드 본체 표면 IPA 세정 후 건조 → (6) 신품 Membrane 장착(균일한 장력으로 고정) → (7) Retaining Ring 재장착.

### 5.3 교체 후 Pressure Hold Test

Membrane 교체 후 반드시 Pressure Hold Test를 수행한다. 절차: (1) 각 Zone에 설정 압력(3.0 psi) 인가 → (2) 공급 밸브 차단 → (3) 30초간 압력 변화 모니터링 → (4) 합격 기준: 30초간 압력 강하 < 0.3 psi (전 Zone). 불합격 시 Membrane 재장착 또는 헤드 본체 Air Channel 재점검.

## 6. Backing Film 교체 절차

### 6.1 교체 판단 기준

Backing Film 교체 판단 기준이다. 분기 PM(캐리어 헤드 오버홀) 시 정기 교체. 접착면 박리 또는 들뜸 발생. 웨이퍼 이면(Backside) 결함 발생. Membrane 교체 시 동시 교체 권장. WIWNU 이상 원인이 Backing Film으로 특정된 경우.

### 6.2 교체 절차

Backing Film 교체 절차이다. (1) 캐리어 헤드에서 Membrane 및 Retaining Ring 분리 → (2) 기존 Backing Film을 헤드 본체에서 제거 → (3) 접착면 잔여물을 IPA로 세정 → (4) 헤드 본체 표면 건조 및 이물질 확인 → (5) 신품 Backing Film을 중심부터 부착(기포 방지) → (6) 롤러로 밀착, 전체 면적 접착 확인 → (7) Membrane, Retaining Ring 재장착 후 Pressure Hold Test.

## 7. 슬러리 타입 변경 절차

### 7.1 Cross-Contamination 방지 원칙

슬러리 타입 변경 시 Cross-Contamination(교차 오염) 방지가 핵심이다. 산성 슬러리(Cu/W)와 알칼리성 슬러리(Oxide)가 혼합되면 입자 응집 및 결함이 발생한다. 원칙: 충분한 라인 플러시(최소 20분), 전용 패드 사용(타입별 패드 분리), 분배 노즐 교체 또는 세정, 전환 후 반드시 Qualification 실행.

### 7.2 Oxide → Cu 슬러리 전환 절차

Oxide 슬러리에서 Cu 슬러리로 전환하는 절차이다. (1) Oxide 슬러리 공급 중단 → (2) DI water 플러시 20분(500 ml/min) → (3) 배출수 pH 확인(중성 근접: pH 6~8) → (4) Oxide 패드 제거, 플래튼 세정 → (5) Cu 전용 패드 부착 → (6) Cu 슬러리 연결 및 프라이밍 → (7) 더미 웨이퍼 5매 처리 → (8) Qualification(MRR, 결함) 실행.

### 7.3 Cu → Oxide 슬러리 전환 절차

Cu 슬러리에서 Oxide 슬러리로 전환하는 절차이다. (1) Cu 슬러리 공급 중단 → (2) DI water 플러시 20분(500 ml/min) → (3) 배출수 pH 확인(중성 근접) → (4) Cu 패드 제거, 플래튼 세정(Cu 잔류물 완전 제거 중요) → (5) Oxide 전용 패드 부착 → (6) Oxide 슬러리 연결 및 프라이밍 → (7) 더미 웨이퍼 5매 처리 → (8) Qualification 실행.

### 7.4 전환 후 검증

슬러리 타입 전환 후 검증 절차이다. (1) Test wafer 3매 연마 → (2) MRR 확인(목표값 ± 10%) → (3) 결함 검사(스크래치, 파티클 기준 이내) → (4) 이전 슬러리 잔류 여부 확인(pH 측정, 파티클 분석) → (5) 합격 시 양산 투입, 불합격 시 추가 플러시 후 재검증. 전환 이력을 장비 로그에 기록한다.
