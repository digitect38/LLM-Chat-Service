# CMP 슬러리 및 패드 교체 절차

## 1. 패드 교체 절차 (SOP-CMP-PAD-001)

### 1.1 교체 판단 기준

| 기준 | 임계값 | 측정 방법 |
|------|--------|----------|
| 사용 시간 | > 500시간 | 장비 로그 |
| 패드 두께 | < 1.0 mm | 두께 게이지 |
| MRR 저하 | > 15% (신품 대비) | Test wafer 측정 |
| WIWNU 악화 | > 5% | Test wafer 측정 |
| 표면 상태 | 심한 glazing, groove 마모 | 육안 검사 |

### 1.2 필요 도구 및 자재
- 신품 polishing pad (IC1000/SubaIV 또는 동등 사양)
- PSA (Pressure Sensitive Adhesive) 접착 패드
- IPA (Isopropyl Alcohol) 세정액
- 클린 와이퍼
- 패드 두께 게이지
- 고무 롤러 (기포 제거용)
- 보호 장갑, 보안경

### 1.3 교체 절차

#### Step 1: 준비 (10분)
1. 장비를 PM 모드로 전환
2. 슬러리 공급 중단
3. DI water로 플래튼 및 패드 세정 (5분)
4. 플래튼 회전 정지
5. 캐리어 헤드를 대기 위치로 이동

#### Step 2: 기존 패드 제거 (15분)
1. 패드 한쪽 모서리에서 PSA 접착면을 분리 시작
2. 천천히 당겨 패드를 플래튼에서 분리
3. **주의**: 급격하게 당기면 PSA 잔여물이 플래튼에 남음
4. 플래튼 표면의 PSA 잔여물을 IPA로 세정
5. 클린 와이퍼로 플래튼 표면 완전 건조
6. 플래튼 표면 이물질 및 손상 확인

#### Step 3: 신품 패드 부착 (15분)
1. 신품 패드의 PSA 보호 필름 제거
2. 패드를 플래튼 중심에 정렬하여 위치
3. 중심에서 바깥 방향으로 롤러를 사용하여 부착
4. **핵심**: 기포가 생기지 않도록 천천히 롤러 적용
5. 패드 에지가 플래튼 에지와 정확히 일치하는지 확인
6. 전체 면적의 접착 상태 확인 (들뜸 없음)

#### Step 4: 패드 초기화 (30분)
1. 컨디셔너를 사용한 패드 표면 초기화 (break-in)
   - Downforce: 7 lbf (일반 운전 대비 높음)
   - 시간: 20분
   - 플래튼 속도: 60 rpm
   - DI water 공급: 200 ml/min
2. Break-in 후 패드 표면 상태 확인
3. 슬러리 공급 시작

#### Step 5: Qualification (30분)
1. 더미 웨이퍼 5매 처리 (패드 안정화)
2. Test wafer 3매 연마 후 측정
   - MRR: 목표값 ± 10% 이내
   - WIWNU: < 5%
   - 스크래치: 없음
3. 측정 결과 기록 및 승인

### 1.4 패드 교체 후 주의사항
- 초기 10매: Zone 압력을 5% 낮게 설정 (새 패드 높은 MRR 보상)
- 초기 50매: 매 10매마다 MRR 추이 모니터링
- 컨디셔닝 레시피: 초기 2일간 강화 컨디셔닝 적용

## 2. 슬러리 교체 절차 (SOP-CMP-SLR-001)

### 2.1 교체 판단 기준

| 기준 | 임계값 | 측정 방법 |
|------|--------|----------|
| 슬러리 잔량 | < 10% | 탱크 레벨 센서 |
| 사용 기간 | > 72시간 (개봉 후) | 개봉 일시 라벨 |
| pH 변화 | 초기값 대비 ±0.5 | pH meter |
| Particle size | D50 기준 ±20% | Particle analyzer |
| MRR 저하 | > 10% | Test wafer 측정 |

### 2.2 슬러리 탱크 교체 절차

#### Step 1: 준비
1. 신품 슬러리 탱크를 최소 30분 전 반입 (온도 안정화)
2. 슬러리 교반 확인 (침전 방지)
3. 탱크 라벨 확인: 슬러리 타입, 로트 번호, 유효 기간

#### Step 2: 기존 슬러리 배출
1. 장비를 idle 상태로 전환 (연마 진행 없음 확인)
2. 슬러리 공급 밸브 차단
3. 슬러리 공급 라인 DI water 플러시 (5분)
4. 기존 탱크 연결 해제
5. 기존 탱크의 잔량 기록 후 폐슬러리 처리

#### Step 3: 신품 슬러리 연결
1. 신품 탱크를 공급 위치에 설치
2. 공급 라인 연결 (퀵 커플링)
3. 연결부 누수 확인
4. 탱크 개봉 일시 라벨 부착
5. 교반기 가동 시작

#### Step 4: 프라이밍 및 검증
1. 슬러리 펌프 프라이밍 (공기 제거)
2. 슬러리 유량 확인 (설정값 ± 5%)
3. 슬러리 pH 측정 → 스펙 이내 확인
4. Test wafer 1매 연마 후 MRR 확인
5. 교체 기록 작성

### 2.3 슬러리 라인 플러시 절차

정기 또는 슬러리 타입 변경 시 수행:

1. 슬러리 공급 중단
2. DI water 밸브 개방
3. DI water 플러시: 500 ml/min × 10분
4. 배출수 확인: 투명해질 때까지 계속
5. 슬러리 타입 변경 시: 추가 플러시 10분
6. 밸브 전환 후 신규 슬러리 공급 시작

### 2.4 슬러리 관리 주의사항
- **온도 관리**: 슬러리 보관 온도 15~25°C
- **유효 기간 엄수**: 개봉 후 72시간 초과 슬러리 사용 금지
- **혼합 금지**: 서로 다른 로트의 슬러리 혼합 금지
- **결정화 방지**: 슬러리 노출 부위 정기 세정
- **폐슬러리 처리**: 환경 규정에 따라 지정 용기에 수집

## 3. 컨디셔너 디스크 교체

### 3.1 교체 판단 기준
- 사용 시간 > 200시간
- 다이아몬드 입자 탈락 (육안 + 현미경 확인)
- Pad conditioning 후에도 MRR 회복 안 됨
- 패드 표면에 비정상 스크래치 패턴

### 3.2 교체 절차
1. 컨디셔너 arm을 정비 위치로 이동
2. 디스크 고정 나사 분리 (3~4개)
3. 기존 디스크 제거 → 사용 시간 기록
4. 마운트 표면 세정 (IPA)
5. 신품 디스크 장착 → 나사 균등 토크로 체결
6. Arm sweep 동작 확인
7. 패드 위에서 break-in conditioning (5분)
8. Test wafer로 MRR 및 스크래치 확인

## 4. Retaining Ring 교체

### 4.1 교체 판단 기준
- 두께 < 1.5 mm (초기 두께 대비)
- 불균일 마모 (편마모)
- 웨이퍼 에지 WIWNU 악화
- 알람 A123 (Head Pressure Out of Range) 반복 발생

### 4.2 교체 절차
1. 캐리어 헤드를 정비 위치로 이동
2. 헤드 하부의 retaining ring 고정 볼트 분리
3. 기존 ring 제거 → 두께 4점 측정 기록
4. 헤드 마운트 면 세정
5. 신품 ring 장착 → 볼트 균등 토크 체결 (15 N·m)
6. Pressure hold test 실행
7. Test wafer 3매 연마 → MRR, WIWNU, 에지 프로파일 확인
