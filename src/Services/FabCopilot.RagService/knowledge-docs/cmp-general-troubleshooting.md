# CMP 일반 트러블슈팅 가이드

## 1. 연마율(MRR) 이상

### 1.1 MRR 저하 (Material Removal Rate Decrease)

연마율이 목표 대비 15% 이상 감소한 상태이다. 공정 시간 증가와 EPD 신호 지연이 동반된다.

| 원인 | 확인 방법 | 조치 |
|------|----------|------|
| Pad glazing | 패드 표면 광택 확인 | Pad conditioning 강화 (downforce +1 lbf) |
| 슬러리 농도 저하 | 슬러리 pH 측정 | 슬러리 탱크 교체 |
| 슬러리 유량 부족 | Flow meter 확인 | 펌프 압력 조정, 필터 교체 |
| 패드 수명 도달 | 사용 시간 확인 | 패드 교체 |
| 온도 이상 | 플래튼 온도 확인 | 냉각수 공급 확인 |
| 압력 설정 오류 | 레시피 파라미터 확인 | 압력 재설정 |

### 1.2 MRR 과다 (Excessive MRR)

연마율이 목표 대비 15% 이상 증가한 상태이다. 과연마(Over-polish)로 막 두께 스펙 이탈이 발생한다.

| 원인 | 확인 방법 | 조치 |
|------|----------|------|
| 새 패드 초기 높은 MRR | 패드 사용 이력 확인 | Break-in 웨이퍼 10매 처리 |
| 압력 과다 설정 | 레시피 확인 | 압력 5~10% 하향 |
| 슬러리 과공급 | Flow rate 확인 | Flow rate 조정 |
| 컨디셔닝 과도 | 컨디셔너 설정 확인 | Downforce 및 sweep 조정 |

## 2. 균일도(WIWNU) 이상

### 2.1 Center-Fast 프로파일

웨이퍼 중심부 연마율이 에지 대비 높은 상태이다. WIWNU가 5%를 초과한다. 원인 및 조치: (1) Zone 1(Center) 압력 과다 → Zone 1 압력 0.3~0.5 psi 감소, (2) Retaining Ring 압력 부족 → Ring 압력 0.5 psi 증가, (3) 슬러리 분배 불균일 → 슬러리 공급 위치 및 유량 확인.

### 2.2 Center-Slow 프로파일

웨이퍼 중심부 연마율이 에지 대비 낮은 상태이다. WIWNU가 5%를 초과한다. 원인 및 조치: (1) Zone 1(Center) 압력 부족 → Zone 1 압력 0.3~0.5 psi 증가, (2) Edge Zone 압력 과다 → Zone 4, 5 압력 0.3 psi 감소, (3) Retaining Ring 과도 마모 → Ring 두께 확인, 1.5mm 미만 시 교체.

### 2.3 Edge-Fast (Edge Erosion)

웨이퍼 에지 3~5mm 영역이 과연마된 상태이다. 에지 프로파일에서 급격한 두께 drop이 발생한다. 원인 및 조치: (1) Retaining Ring 마모 → Ring 두께 측정, 교체 검토, (2) Edge Zone 압력 과다 → Zone 5 압력 감소, (3) Ring 압력 부족 → Retaining Ring 압력 1.0 psi 증가.

### 2.4 M-Shape/W-Shape 프로파일

M-Shape은 중간 Zone(Zone 3) 연마율이 높은 패턴이다. Zone 3 압력을 0.2 psi 낮추어 조정한다. W-Shape은 중간 Zone 연마율이 낮은 패턴이다. Zone 3 압력을 0.2 psi 높인다. 두 패턴 모두 슬러리 분배 상태와 컨디셔너 Sweep 패턴을 확인한다.

## 3. 스크래치 (Scratch)

### 3.1 Micro-Scratch

현미경으로만 관찰 가능한 미세 스크래치이다. 웨이퍼 전면에 분포한다.

| 원인 | 확인 방법 | 조치 |
|------|----------|------|
| 슬러리 내 대형 입자 | Particle count | 슬러리 교체, 필터 교체 |
| 패드 표면 이물질 | 패드 표면 검사 | Pad conditioning + 세정 |
| 컨디셔너 디스크 이상 | 디스크 표면 검사 | 디스크 교체 |

### 3.2 Macro-Scratch

육안으로 관찰 가능한 깊은 스크래치이다. 특정 방향 또는 패턴으로 발생한다.

| 원인 | 확인 방법 | 조치 |
|------|----------|------|
| 패드 groove에 이물질 | 패드 표면 검사 | 이물질 제거 또는 패드 교체 |
| Retaining ring 파손 | Ring 표면 검사 | 즉시 교체 |
| 웨이퍼 칩핑 잔재 | 이전 웨이퍼 상태 확인 | 패드 세정 + 더미 처리 |
| 슬러리 건조/결정화 | 슬러리 라인 점검 | 라인 플러시 |

## 4. 디싱/에로전 (Dishing & Erosion)

### 4.1 Cu Dishing

Cu 배선 영역이 주변 산화막 대비 오목하게 연마된 상태이다. Wide line에서 더 심하게 발생한다. 조치: (1) 과연마(Over-polish) → EPD 후 over-polish 시간 5초 단위 단축, (2) 압력 과다 → 전체 압력 0.5 psi 감소, (3) 슬러리 선택비 부족 → 슬러리 타입 변경 검토.

### 4.2 Oxide Erosion

Dense pattern 영역의 산화막이 과도하게 연마된 상태이다. 패턴 밀도에 따라 두께 차이가 발생한다. 조치: (1) 과연마 → Over-polish 시간 최소화, (2) 슬러리 선택비 → High-selectivity 슬러리 사용, (3) 패드 강성 → 하드 패드 사용 검토.

## 5. 장비 알람 대응

### 5.1 주요 알람 코드 요약

CMP 장비의 주요 알람 코드 요약이다. 상세 대응은 cmp-alarm-code-reference 문서를 참조한다.

| 알람 코드 | 설명 | 심각도 | 즉시 조치 |
|----------|------|--------|----------|
| A100 | Emergency Stop | Critical | 안전 확인 후 리셋 |
| A101 | Vacuum 이상 | Critical | 척킹 상태 확인 |
| A110 | Platen 과부하 | High | 패드/슬러리 확인 |
| A120 | 슬러리 유량 이상 | High | 라인/펌프 점검 |
| A123 | Head Pressure 이상 | High | 헤드 점검 |
| A140 | EPD 미감지 | Medium | 센서 점검 |

### 5.2 알람 대응 기본 원칙

알람 심각도별 대응 원칙이다. Critical 알람: 즉시 장비 정지, 웨이퍼 안전 확보, 엔지니어 호출. High 알람: 진행 중 웨이퍼 완료 후 장비 정지, 원인 분석. Medium 알람: 로그 확인 후 조건부 계속 운전 가능, 교대 내 조치 완료.

## 6. 웨이퍼 파손 대응

### 6.1 연마 중 웨이퍼 파손

연마 중 웨이퍼가 파손된 경우의 대응 절차이다. (1) 즉시 E-Stop → (2) 캐리어 헤드 상승 → (3) 패드 위 파편 회수(진공 피커 사용) → (4) 패드 표면 검사, 파편 잔여 시 패드 교체 → (5) 슬러리 라인 DI water 플러시 10분 → (6) 더미 웨이퍼 3매 처리 후 Qualification → (7) 사고 보고서 작성.

### 6.2 예방 조치

웨이퍼 파손 예방 수칙이다. 웨이퍼 로딩 시 척킹 센서 정상 확인. Retaining ring 두께 정기 점검(기준 > 2.0mm). 과도한 압력 설정 방지(Zone 압력 < 6.0 psi). 비정상 torque 발생 시 즉시 정지. EPD 미감지 시 최대 시간 제한으로 과연마 방지.

## 7. Head Zone 압력 진동 (Pressure Oscillation)

### 7.1 증상 및 원인

Zone 7 또는 Zone 8에서 압력이 0.5~2Hz 주기로 진동하는 현상이다. WIWNU 증가와 MRR 불안정이 동반된다. 원인: (1) Pad Glazing — 패드 표면 경화로 마찰 계수 변화, (2) Retaining Ring 마모 — 에지 압력 불균형, (3) Membrane Leak — 미세 누수로 압력 제어 불안정.

### 7.2 조치 방법

압력 진동 발생 시 조치 절차이다. (1) Pad conditioning 프로그램 실행(60초, 5 lbf downforce) → (2) Test wafer로 연마율 확인 → (3) 개선 안 되면 패드 교체 → (4) Retaining ring 두께 측정(기준: > 2.0mm, 미달 시 교체) → (5) Membrane 누수 점검(Pressure Hold Test 실행, 30초간 압력 강하 < 0.3 psi).

## 8. 로봇/이송 문제

### 8.1 웨이퍼 미스핸들링 (Wafer Mishandling)

로봇이 웨이퍼를 정상적으로 픽업/플레이스하지 못하는 문제이다. 증상: 웨이퍼 센서 미감지(A151), 웨이퍼 위치 어긋남, 웨이퍼 떨어짐. 원인: 진공 흡착 불량, 로봇 티칭 오프셋, 로봇 암 그리퍼 오염. 조치: 진공 라인 점검, 그리퍼 세정, 티칭 포인트 재확인(허용 오차 ±0.5mm).

### 8.2 FOUP 도어 이상 (FOUP Door Malfunction)

FOUP 도어 개폐가 정상적으로 동작하지 않는 문제이다. 증상: 알람 A152, 도어 미개방/미폐쇄, FOUP 인식 불가. 원인: FOUP 미정위치 장착, 도어 래치 기구 고장, 도어 모터 이상, FOUP 키 불일치. 조치: FOUP 재장착 시도, 도어 래치 수동 동작 확인, Load Port 초기화, 다른 FOUP으로 교체 테스트.

### 8.3 로봇 티칭 오류 복구 (Robot Teaching Recovery)

로봇이 정확한 위치에 도달하지 못하는 문제이다. 알람 A150이 반복 발생한다. 원인: 진동/충격으로 티칭 포인트 이동, 기구부 마모로 위치 편차, 엔코더 이상. 복구 절차: (1) 로봇을 수동 모드로 전환 → (2) 각 스테이션 티칭 포인트 확인 → (3) 편차 > 0.5mm인 포인트 재티칭 → (4) 더미 웨이퍼로 전체 이송 경로 테스트 → (5) 정상 확인 후 자동 모드 전환.

## 9. EPD 관련 문제

### 9.1 EPD 미감지 상세 대응 (EPD Not Detected)

설정 최대 연마 시간 내에 EPD 종점 신호가 감지되지 않은 상태이다. 알람 A140이 발생하며 시간 기반 백업으로 전환된다. 원인: EPD 센서 오염, 입력 막 두께 변동(두꺼움), EPD 임계값 과도 설정, 슬러리 잔여물로 센서 차폐. 조치: 센서 광학창 세정, 임계값 완화(변화율 5% → 3%), Reference wafer로 검증.

### 9.2 EPD 오감지 (False Trigger)

실제 종점 도달 전에 EPD 신호가 발생하는 문제이다. 연마 부족(Under-polish)이 발생한다. 원인: EPD 임계값 과도하게 낮음, 노이즈 신호, 막질 변화로 반사율 급변, 진동에 의한 센서 오차. 조치: 임계값 상향(변화율 3% → 5%), 이동 평균 필터 적용(3~5 포인트), 센서 마운트 진동 차단 확인.

### 9.3 EPD 신호 불안정 (EPD Signal Instability)

EPD 신호에 노이즈가 심하여 종점 판정이 어려운 상태이다. 원인: 센서 케이블 접촉 불량, 전기적 노이즈, 광학창 부분 오염, 슬러리 잔류물. 조치: 센서 케이블 커넥터 재결합, 접지 상태 확인, 광학창 세정, 신호 필터 파라미터 조정(이동 평균 윈도우 확대).

## 10. 트러블슈팅 의사결정 트리

### 10.1 MRR 이상 시 의사결정 흐름

MRR 이상 감지 시 다음 순서로 점검한다. (1) 슬러리 확인(유량, pH, 잔량) → 이상 시 슬러리 조치 → (2) 패드 확인(glazing, 두께, 사용시간) → 이상 시 컨디셔닝 또는 패드 교체 → (3) 압력 확인(Zone별 실제 압력 vs 설정값) → 이상 시 Pressure Hold Test → (4) 온도 확인(플래튼, 슬러리) → 이상 시 냉각수 점검. 모두 정상이면 레시피 변경 이력 확인.

### 10.2 스크래치 발생 시 의사결정 흐름

스크래치 감지 시 다음 순서로 점검한다. (1) 스크래치 패턴 확인(방사형/동심원/랜덤/에지) → (2) 방사형이면 컨디셔너 디스크 검사 → (3) 랜덤이면 슬러리 particle count 확인 → (4) 에지 집중이면 Retaining Ring 검사 → (5) 패드 표면 이물질 확인 → (6) 슬러리 라인 결정화 여부 점검. 원인 특정 후 해당 부품 교체/세정.

### 10.3 균일도 이상 시 의사결정 흐름

WIWNU 이상 감지 시 다음 순서로 점검한다. (1) 두께 프로파일 확인(Center-Fast/Slow, Edge-Fast, M/W-Shape) → (2) 해당 Zone 압력 조정(0.2~0.3 psi 단위) → (3) Retaining Ring 두께 측정(< 2.0mm면 교체 검토) → (4) Membrane Pressure Hold Test → (5) 컨디셔너 Sweep 패턴 확인 → (6) 슬러리 분배 균일성 확인.

### 10.4 알람 반복 발생 시 의사결정 흐름

동일 알람이 3회 이상 반복 발생 시 대응 절차이다. (1) 알람 이력 로그 분석(발생 시점, 간격, 조건) → (2) 단순 리셋으로 해결되는지 확인 → (3) 센서 자체 이상 여부 점검(교정 상태, 출력값) → (4) 해당 서브시스템 하드웨어 점검 → (5) 근본 원인 분석(Root Cause Analysis) → (6) 시정 조치 후 모니터링 기간 설정(최소 24시간 또는 50매).

## 11. Slurry Flow Rate 이상

### 11.1 증상 및 원인

슬러리 유량이 설정값 대비 20% 이상 감소하는 문제이다. MRR 급격한 저하와 Torque 값 상승이 동반된다. 원인: (1) 슬러리 공급 라인 결정화에 의한 막힘, (2) 펌프 이상(다이어프램 마모), (3) 필터 막힘(차압 > 10 psi), (4) 슬러리 탱크 잔량 부족.

### 11.2 조치 방법

슬러리 유량 이상 시 조치 절차이다. (1) 슬러리 탱크 잔량 확인 → (2) 공급 라인 DI water 플러시(5분) → (3) 필터 교체(PM 주기 확인) → (4) 펌프 출력 압력 확인(정상: 15~25 psi) → (5) 유량 복구 확인 후 Test wafer로 MRR 검증.
