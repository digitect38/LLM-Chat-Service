# CMP 장비 안전 절차

## 1. 일반 안전 수칙

### 1.1 PPE 착용 기준 (Personal Protective Equipment)

CMP 장비 작업 시 필수 PPE는 다음과 같다. 클린룸 방진복(Bunny Suit), 안전화(정전기 방지), 보안경(슬러리 비산 방지), 내화학 장갑(니트릴 재질), 필요 시 방독 마스크(화학물질 취급 시). 슬러리 교체, 세정액 취급, PM 작업 시에는 내화학 앞치마를 추가 착용한다.

### 1.2 클린룸 안전 행동 수칙

클린룸 내에서는 뛰지 않는다. 장비 동작 중 내부에 손을 넣지 않는다. 비상 정지 버튼 위치를 사전에 숙지한다. 장비 커버가 열린 상태에서 가동하지 않는다. 바닥에 슬러리나 DI water가 흘린 경우 즉시 닦아 미끄럼을 방지한다. 1인 작업 금지 원칙을 준수한다(야간/휴일 PM 시 2인 1조).

### 1.3 작업 전 안전 확인 (Pre-Work Safety Check)

작업 시작 전 다음 사항을 확인한다. 안전 교육 이수 여부, PPE 착용 상태, 장비 E-Stop 버튼 위치 및 동작 확인, 배기 시스템 정상 가동, 화학물질 MSDS 비치 여부, 비상 샤워/세안기 위치 확인. 점검 결과를 작업 전 체크리스트에 기록하고 서명한다.

## 2. 화학물질 안전

### 2.1 슬러리 취급 안전 (Slurry Handling Safety)

CMP 슬러리는 연마 입자(SiO2, Al2O3, CeO2)와 화학 첨가제를 포함한다. Oxide 슬러리는 pH 10~11(강알칼리), Cu 슬러리는 pH 3~4(산성)이다. 피부/눈 접촉 시 즉시 15분간 흐르는 물로 세척한다. 흡입 방지를 위해 환기가 확보된 곳에서 취급한다. MSDS를 반드시 숙지하고 작업장에 비치한다.

### 2.2 세정 화학물질 안전 (Cleaning Chemical Safety)

Post-CMP 세정에 사용되는 화학물질의 안전 정보이다. IPA(이소프로필 알코올)는 인화성이므로 화기 엄금, 환기 필수. NH4OH(암모니아수)는 자극성 가스를 발생시키므로 밀폐 공간에서 사용 금지. HF(불산)는 극독성이며, 전용 PPE(HF 전용 장갑, 보안경) 착용 필수. HF 노출 시 즉시 칼슘 글루코네이트 겔 도포 후 응급 이송.

### 2.3 화학물질 유출 대응 (Chemical Spill Response)

화학물질 유출 시 즉각 대응 절차이다. 소량 유출(< 500 ml): 적절한 PPE 착용, 흡착재(스필 키트)로 흡수, 지정 폐기물 용기에 수거. 대량 유출(> 500 ml): 해당 구역 인원 대피, 환경안전팀 즉시 연락, 배기 가동 확인, 유출 확산 방지를 위해 주변 배수구 차단. 유출 물질, 양, 시간, 대응 내용을 기록한다.

### 2.4 폐액 처리 (Waste Disposal)

CMP 공정에서 발생하는 폐액은 종류별로 분리 수거한다. 폐슬러리는 전용 수거 용기(라벨: 폐슬러리)에 수집한다. 산성 폐액과 알칼리성 폐액을 절대 혼합하지 않는다. 폐액 용기가 80% 차면 환경안전팀에 수거 요청한다. 폐액을 일반 배수구에 버리는 것은 환경법 위반이다.

## 3. 전기 안전

### 3.1 LOTO 절차 상세 (Lock-Out/Tag-Out Procedure)

LOTO는 장비 에너지를 차단하고 잠금/표지를 설치하여 작업자 안전을 보장하는 절차이다. 절차: (1) 장비 정지 → (2) 에너지 차단(전원 차단기 OFF) → (3) 잔류 에너지 확인(콘덴서 방전, 잔압 해소) → (4) 개인 잠금 장치(자물쇠) 설치 → (5) 작업자 이름/일시가 기재된 태그 부착 → (6) 에너지 차단 확인(시동 시도). 작업 완료 후 역순 해제.

### 3.2 전기 배선 점검 안전 (Electrical Inspection Safety)

전기 배선 점검 시 반드시 LOTO를 실행한 후 작업한다. 절연 공구를 사용한다. 콘덴서 방전 확인 후 작업한다(대기 시간 최소 5분). 멀티미터로 무전압 확인 후 작업한다. 젖은 손으로 전기 부품을 만지지 않는다. 자격이 없는 인원은 전기 작업을 하지 않는다.

### 3.3 비상 전원 차단 (Emergency Power Shutdown)

비상 전원 차단이 필요한 상황: 감전 사고, 전기 화재, 장비 이상 발열. 차단 방법: (1) 장비 E-Stop 버튼 → (2) 장비 메인 차단기(MCC) OFF → (3) 필요 시 Fab 전력팀에 상위 차단 요청. E-Stop은 소프트웨어적 정지이므로 전기 작업 시에는 반드시 차단기를 물리적으로 OFF 한다.

## 4. 기계 안전

### 4.1 회전 부품 안전 (Rotating Parts Safety)

CMP 장비의 주요 회전부는 Platen(60~120 rpm), Carrier Head(50~100 rpm), Conditioner Arm이다. 회전 중 접촉 시 심각한 부상(끼임, 절단)이 발생할 수 있다. 안전 수칙: 장비 가동 중 내부 접근 금지, 인터록 해제 상태에서 회전부 가동 금지, 느슨한 옷/장갑이 회전부에 감기지 않도록 주의.

### 4.2 끼임 방지 (Pinch Point Prevention)

끼임 위험 부위: 캐리어 헤드 승하강 영역, 로봇 암 동작 범위, FOUP 도어 개폐부, 컨디셔너 Arm 스윕 범위. 예방 수칙: 장비 동작 반경에 신체를 넣지 않는다. PM 작업 시 반드시 장비를 정지하고 LOTO를 실행한다. 인터록 센서가 정상 동작하는지 정기적으로 확인한다.

### 4.3 중량물 취급 (Heavy Object Handling)

CMP 장비의 중량 부품: 캐리어 헤드(약 15~25 kg), 플래튼(교체 시), 슬러리 탱크(만충전 시 20~30 kg). 취급 수칙: 20 kg 이상 부품은 2인 이상 취급 또는 호이스트 사용. 허리를 굽히지 않고 무릎을 구부려 들어올린다. 운반 경로의 장애물을 사전에 제거한다.

## 5. 비상 대응

### 5.1 Emergency Stop 사용법

E-Stop 버튼은 장비 전면과 후면에 각 1개 이상 설치되어 있다. 버튼을 강하게 누르면 장비의 모든 동작이 즉시 중단된다. 사용 상황: 웨이퍼 이탈 감지, 이상 소음/진동, 화학물질 누출, 인체 접촉 위험 발생 시. 리셋: 원인 해결 후 E-Stop 버튼을 시계 방향으로 돌려 해제, 소프트웨어에서 알람 클리어, 장비 홈 복귀 실행.

### 5.2 화재 대응 (Fire Response)

CMP 장비 화재 대응 절차이다. (1) 즉시 E-Stop 및 전원 차단 → (2) 화재 경보 발령 → (3) 소규모 화재 시 CO2 소화기로 초기 진화 시도(전기 화재에 물 사용 금지) → (4) 진화 불가 시 즉시 대피 → (5) Fab 방재팀/119 신고. 대피 시 클린룸 출입문을 닫아 연기 확산을 방지한다.

### 5.3 지진 대응 (Earthquake Response)

지진 발생 시 CMP 장비 대응 절차이다. (1) 장비 E-Stop 즉시 실행 → (2) 화학물질 유출 여부 확인 → (3) 진동이 멈출 때까지 장비에서 이격, 안전한 곳에 대기 → (4) 진동 종료 후 장비 상태 점검(배관 누수, 기구부 변형, 수평도) → (5) 이상 없음 확인 후 단계적 재가동. 슬러리 라인 누출 여부를 반드시 확인한다.

### 5.4 정전 대응 (Power Outage Response)

정전 시 CMP 장비 대응 절차이다. 연마 중 정전 시 웨이퍼가 패드 위에 남아있을 수 있다. (1) 전원 복구 후 장비 상태 확인 → (2) 패드 위 웨이퍼 회수(수동 모드) → (3) 슬러리 라인 건조/결정화 방지를 위해 DI water 플러시 → (4) 장비 초기화 및 홈 복귀 → (5) Qualification 실행 후 생산 재개.

### 5.5 사고 보고 절차 (Incident Reporting)

사고(인적/물적) 발생 시 보고 절차이다. (1) 부상자 응급 처치 및 안전 확보 → (2) 현장 보존(사진 촬영) → (3) 즉시 상사 및 환경안전팀에 구두 보고 → (4) 24시간 이내 사고 보고서 작성(일시, 장소, 경위, 피해, 조치) → (5) 재발 방지 대책 수립(CAPA). 경미한 아차 사고(Near Miss)도 반드시 보고한다.

## 6. 안전 교육

### 6.1 필수 교육 목록

CMP 장비 작업자가 이수해야 할 안전 교육 목록이다. 클린룸 일반 안전(입사 시), 화학물질 취급(MSDS/GHS, 연 1회), LOTO 교육(연 1회), CMP 장비 안전 교육(신규 배치 시), 비상 대응 훈련(연 2회), 소화기 사용법(연 1회), HF 안전(불산 취급 시, 연 1회).

### 6.2 교육 갱신 주기

안전 교육의 갱신 주기이다. 법정 의무 교육(산업안전보건법): 정기 교육 분기 6시간 이상. 화학물질 취급 교육: 연 1회 이상 갱신. LOTO 인증: 연 1회 재인증. 비상 대응 모의 훈련: 연 2회(상반기/하반기). 교육 미이수 작업자는 장비 작업 투입이 불가하다. 교육 이수 기록은 3년간 보관한다.
